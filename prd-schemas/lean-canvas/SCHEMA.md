A lean, one-page PRD format adapted from Ash Maurya's Lean Canvas. Best for early-stage products, MVPs, and rapid validation.

## 10. Agent Prompt Guide

When generating a Lean Canvas PRD:
1. **One problem → one solution mapping.** Each feature must trace to a specific problem.
2. **UVP must be a single sentence.** If it takes a paragraph, it's not clear enough.
3. **Metrics must be actionable.** "Revenue" is not a metric; "MRR" is.
4. **Be honest about unfair advantage.** If you don't have one yet, say so — don't fabricate.
5. **Customer segments must distinguish user from buyer.** B2B products often have different users and buyers.
